# Workshop Esthétiques algorithmiques (semaine du 12/01/2026)

---

Mon projet préféré est la Pokéfourmi de Langton (projet "Free"), car il mélange l'atelier que j'ai le plus aimé : l'automate cellulaire (dans mon cas : la fourmi de Langton) et ma licence préférée, Pokémon.

Il n'y a pas d'installation particulière : j'ai utilisé l'installation de base de Processing. Pour le code Python de l'algorithme du poème d'amour, j'ai utilisé PyCharm de JetBrains. Normalement, il n'y a aucun souci pour le faire fonctionner à ce niveau-là.\
J'indique dans les différents README, dans chaque sous-répertoire, quelle est la version que je rends, dans le cas où j'y ai laissé différentes expérimentations.
