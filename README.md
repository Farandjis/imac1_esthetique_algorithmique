Mon projet préféré est la Pokéfourmi de Langton (projet "Free") car il mélange l'atelier que j'ai le plus aimé : l'automate cellulaire (dans mon cas : la fourmi de langton) et ma licence préféré Pokémon.

J'indique dans les différents README dans chaque sous répertoire quelle est la version que je rends dans le cas où j'y ai laissé différentes expérimentations.
