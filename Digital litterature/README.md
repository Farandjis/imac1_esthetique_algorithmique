## <a id="p1"></a> I - Atelier 4.1 : l'algorithme de Strachey

### 1. Récriture de l'algorithme
J'ai récupéré le script Python de Nick Montfort https://nickm.com/memslam/love_letters.py et traduit les mots en français avec ChatGPT.\
J'ai essayé de réajuster les tournures de phrase pour qu'elle soit plus française, mais aussi de faire la distinction entre le féminin/masculin pour mon/ma.


### 2. Changer le thésaurus
J'ai changé les différents mots par des mots venant de lettres de motivation afin d'en faire un générateur de lettre de motivation.\
Pour le plaisir, j'ai remplacé certains mots dans les listes par des mots comme déplacés : "haineux", "coquin", "crever", "généré par ChatGPT".\
\
